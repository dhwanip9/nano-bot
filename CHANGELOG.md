# Changelog

## v1.0.0 — Initial release

### What's in this release
- Floating desktop companion for Mac, Windows, and Linux
- 20 blindspot categories covering architecture, execution, deployment, and maintenance
- Paste-to-scan: paste any prompt or Claude output for instant analysis
- Auto-watch clipboard: scans automatically when you copy something new
- 2-exchange chat with handoff prompt generation
- Session memory with auto-summarization every 5 exchanges
- Tray icon with new session and hide controls
- Auto-updater — future releases install silently in the background
